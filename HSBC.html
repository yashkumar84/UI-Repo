Key Concept: Complement Logic

Agar target = 9 aur current = 2, toh complement = 7 dhundo HashMap mein!

Java
⏱ Time: O(n) 💾 Space: O(n)
import java.util.HashMap; public int[] twoSum(int[] nums, int target) { HashMap<Integer, Integer> map = new HashMap<>(); for(int i = 0; i < nums.length; i++) { int complement = target - nums[i]; // Check if complement exists if(map.containsKey(complement)) { return new int[]{map.get(complement), i}; } // Store current number with index map.put(nums[i], i); } return new int[]{}; }

✅ Why HashMap?

O(1) average lookup time - instant checking!

🎯 Trade-off

Space badhta hai but time drastically reduce

🌟 Real Use

Netflix recommendations, similar items finding

Three Pointer Technique

Har step mein: Save next → Reverse link → Move forward. Chain mat todna!

Java - Iterative Approach
⏱ Time: O(n) 💾 Space: O(1)
class ListNode { int val; ListNode next; ListNode(int val) { this.val = val; } } public ListNode reverseList(ListNode head) { ListNode prev = null; ListNode current = head; while(current != null) { // Step 1: Save next node (CRITICAL!) ListNode nextTemp = current.next; // Step 2: Reverse current link current.next = prev; // Step 3: Move pointers forward prev = current; current = nextTemp; } return prev; // New head }

Q1: Kadane's Algorithm - Maximum Subarray Sum

Problem: Continuous subarray mein maximum sum find karo

Example: [-2,1,-3,4,-1,2,1,-5,4] → 6 (subarray: [4,-1,2,1])

🔗 LeetCode Link

Approach: Dynamic Programming

Har position pe decide: current ko add karu ya nayi subarray shuru karu?

Formula: maxEndingHere = max(nums[i], maxEndingHere + nums[i])

Java
⏱ O(n) 💾 O(1)
public int maxSubArray(int[] nums) { int maxSoFar = nums[0]; int maxEndingHere = nums[0]; for(int i = 1; i < nums.length; i++) { maxEndingHere = Math.max(nums[i], maxEndingHere + nums[i]); maxSoFar = Math.max(maxSoFar, maxEndingHere); } return maxSoFar; }
Q2: Detect Cycle in Linked List (Floyd's Algorithm)

Concept: Tortoise & Hare - do pointers se cycle detect karo

Logic: Slow 1 step, Fast 2 steps - agar cycle hai toh milenge!

🔗 LeetCode Link

Why It Works?

Circular track analogy - fast runner eventually slow ko catch kar lega!

Java
⏱ O(n) 💾 O(1)
public boolean hasCycle(ListNode head) { if(head == null) return false; ListNode slow = head; ListNode fast = head; while(fast != null && fast.next != null) { slow = slow.next; // 1 step fast = fast.next.next; // 2 steps if(slow == fast) { return true; // Cycle detected! } } return false; }
Q3: Valid Parentheses

Problem: Brackets properly balanced hain ya nahi check karo

Example: "{[()]}" → Valid | "{[(])}" → Invalid

🔗 LeetCode Link

Stack Magic

Opening bracket → Push | Closing bracket → Check & Pop

Application: Every IDE uses this for syntax checking!

Java
⏱ O(n) 💾 O(n)
import java.util.Stack; public boolean isValid(String s) { Stack<Character> stack = new Stack<>(); for(char c : s.toCharArray()) { if(c == '(' || c == '{' || c == '[') { stack.push(c); } else { if(stack.isEmpty()) return false; char top = stack.pop(); if((c == ')' && top != '(') || (c == '}' && top != '{') || (c == ']' && top != '[')) { return false; } } } return stack.isEmpty(); }
Q4: Binary Search (Recursive)

Concept: Sorted array mein O(log n) time mein search

Trick: Middle check → left ya right half recurse

🔗 LeetCode Link

Overflow Safe Formula

❌ mid = (left + right) / 2 → Can overflow!

✅ mid = left + (right - left) / 2 → Safe approach

Java
⏱ O(log n) 💾 O(log n)
public int binarySearch(int[] arr, int target, int left, int right) { if(left > right) return -1; // Overflow safe mid calculation int mid = left + (right - left) / 2; if(arr[mid] == target) return mid; if(arr[mid] > target) { return binarySearch(arr, target, left, mid - 1); } return binarySearch(arr, target, mid + 1, right); }
Q5: Move Zeros to End

Problem: Sabhi zeros ko end mein move karo, order maintain karke

Example: [0,1,0,3,12] → [1,3,12,0,0]

🔗 LeetCode Link

Two Pointer Magic

Ek pointer non-zero track kare - swap karte jao!

Java
⏱ O(n) 💾 O(1)
public void moveZeroes(int[] nums) { int nonZero = 0; for(int i = 0; i < nums.length; i++) { if(nums[i] != 0) { // Swap karo int temp = nums[i]; nums[i] = nums[nonZero]; nums[nonZero] = temp; nonZero++; } } }
