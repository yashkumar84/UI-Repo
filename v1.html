What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that provides low-level utility classes to build custom designs directly in your HTML. Instead of writing custom CSS, you compose designs using pre-built utility classes.

Traditional CSS vs Tailwind CSS

‚ùå Traditional CSS 

/* CSS File */
.custom-button {
    background-color: blue;
    color: white;
    padding: 12px 24px;
    border-radius: 8px;
    font-weight: bold;
}

/* HTML */
<button class="custom-button">
    Click Me
</button>

‚úÖ Tailwind CSS 

/* No CSS File Needed! */



/* HTML Only */
<button class="bg-blue-500 text-white px-6 py-3 rounded-lg font-bold">
    Click Me
</button>

How to Include Tailwind CSS

<script src="https://cdn.tailwindcss.com"></script>

üì± Responsive Design

Tailwind uses mobile-first breakpoints to create responsive designs.

Breakpoint Prefixes

sm: - Small devices (640px and up)
md: - Medium devices (768px and up)
lg: - Large devices (1024px and up)
xl: - Extra large (1280px and up)
2xl: - 2X Extra large (1536px and up)

How to Use Breakpoints

<!-- This div will be:
 - Full width on mobile
 - Half width on medium screens
 - One third width on large screens -->
